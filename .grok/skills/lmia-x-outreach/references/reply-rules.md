# リプライ文案ルール

`generate-outreach.ts` の `buildReply()` およびエージェントの手動調整で遵守。

## 使わない表現

| 表現 | 理由 |
|------|------|
| イケメンホスト | リプライは客も見る。業種イメージと齟齬が出やすい |
| 個人で作ってて / 個人開発 / 個人プロジェクト | 信頼性を損なう。サービス名「エルミア」で十分 |
| 無理なら全然OKです | 締めは下記の定型に統一 |

## 定型

- **締め**: `いかがでしょうか？🙏`
- **サービス名**: `エルミア`（括弧付きで1回触れる）
- **トーン**: 同業への感想依頼。押し売りしない

## 本文バリエーション（自動生成）

intro は3パターンをローテ。エージェント調整時もこの語彙を維持する。

1. ソープで働く方向けの相談チャット（エルミア）を運営してて、同業の方に試してもらってるんです。
2. ソープ嬢向けの相談チャット（エルミア）の感想を聞きたくてリプしました。
3. ソープ嬢向けの相談チャットで、指名・接客・メンタルみたいな話を相談できるツールです。

## 初回リプライ

- URL を入れない（[anti-ban-rules.md](./anti-ban-rules.md)）
- 相手のポストに触れた opener を優先（出勤 / 日記 / 突然すみません）
