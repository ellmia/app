## 1. 現状

### プロダクト

**コアジョブ** — 夜の仕事（ソープ）で「今日から試せる具体アドバイス」を、匿名・低コストで得る

| 項目 | 内容 |
|---|---|
| UI | フルスクリーンチャット（携帯特化） |
| 差別化資産 | イケメンホスト SYSTEM_PROMPT（業界特殊性・安全原則・小さく実験） |
| 技術 | OpenRouter `openrouter/free`、12 msg / 5 min の IP レートリミット |
| オンボーディング | 例示質問 30 件（写メ・指名・クレーム・メンタル等） |
| 成熟度 | **MVP**（2026-06-14 実装完了） |

### トラクション

> 実測値なし。**計測開始が最優先。**

| 指標 | 現在 | 3ヶ月後目標 |
|---|---|---|
| MAU / DAU | 未計測 | MAU 200 / DAU 30 |
| Retention D7 / D30 | 未計測 | ≥ 20% / ≥ 8% |
| セッション msg 数（中央値） | 未計測 | ≥ 4 |
| 有料 / MRR | ¥0 | Phase 1 は ¥0 |
| NPS | 未計測 | ≥ 30 |

### 学び（事実・仮説・リスク）

| 区分 | 内容 |
|---|---|
| **強み** | 例示質問が実務課題と一致。プロンプトが ChatGPT と差別化。運用コスト極小 |
| **想定リスク** | 「先輩の方が信頼できる」／無料 LLM の品質ブレ／プライバシー不安／レートリミット |
| **設計上の発見** | メンタルアプリではなく**集客・単価・方向性**寄り。履歴なしはリテンションとトレードオフ |

## 3. 誰に届けるか（ICP）

### Phase 1 のターゲット

| 項目 | 内容 |
|---|---|
| 誰 | ソープランド在籍、入店 **6〜24 ヶ月**、新人〜中堅 |
| 状況 | 指名・新規・写メ・常連化で伸び悩み。先輩に相談しづらい |
| 代替手段 | 先輩・Girls Heaven 等の記事・ChatGPT・掲示板・Lancers コンサル |
| 予算（仮説） | 月 ¥500〜1,500 なら検討可。まず無料で習慣化 |

### 狙わない

- ベテラン（3 年+、型確立済み）
- メンタル専用ニーズのみ
- 店舗オーナー・マネージャー（B2B は Phase 3 以降）

### ペルソナ「あかり」（入店 1 年）

- **日常** — 写メ更新 → 接客 → 指名待ち → 同僚との距離感
- **苦痛** — 新規が来ない焦り。先輩は「根性」系の返答
- **情報源** — 匿名 X、ガールズヘブン、同僚の写メ

### Super-user（仮説）

- 入店 6〜18 ヶ月、**指名 or 新規**に具体課題
- シフト合間 3〜5 分の短い相談を繰り返す
- 「今日試すこと」が返ると満足度が跳ねる
- **発見場所** — 業界メディアのコメント欄、匿名 X/note、写メネタ困り投稿

## 4. ポジショニング

### 競合マップ

| 種別 | 例 |
|---|---|
| 直接 | ChatGPT / Claude / Gemini、Lancers 人間コンサル |
| 間接 | 店の先輩、Girls Heaven・バニラ記事、掲示板、Awarefy（メンタルのみ競合） |

### 差別化（1 行ずつ）

| Hostorch | 競合の弱点 |
|---|---|
| ソープ特殊性を前提にしたプロンプト | ChatGPT は汎用ビジネス論 |
| 男性視点 × 女性の味方ペルソナ | 先輩は女性視点のみ、記事は一方的 |
| 「今週の最小実験」まで落とす | 記事は情報止まり、AI は抽象論 |
| 匿名・無料・即時・モバイル | 人間コンサルは高単価・予約必要 |

### ポジショニング文

> **指名・新規に悩むソープ嬢**向けの**業界特化 AI キャリア相談チャット**。  
> **ChatGPT や店の先輩**と違い、**夜の仕事の現実を知る男性視点で、今日試せる具体アクションまで一緒に設計**。  
> シフト合間に、写メ・接客・方向性の迷いを低コストで整理できる。

### 訴求メッセージ

- **主軸** — 「イケメンホストに、指名の悩みを 3 分で相談」
- **サブ** — ① 一般論にならない ② 匿名・無料・登録不要 ③ 今週の小さな実験まで一緒に

## 5. Pivot 判断

### 残課題

① 誰が最も熱狂するか不明　② 計測なし　③ 無料モデル品質　④ 収益化未定　⑤ 獲得チャネル未定

### 候補と採用

| 候補 | 判定 | 理由 |
|---|---|---|
| A. 計測・獲得に集中 | **採用** | コアを崩さず検証できる |
| B. 新人〜中堅に特化 | **採用** | メッセージが鋭くなる |
| C. 訴求を指名・新規へ | **採用** | プロンプト設計と一致 |
| D. 風俗全般へ拡大 | 保留 | 専門性が薄まる |
| E. 店舗 B2B SaaS | 却下 | MVP と別物、工数大 |
| F. 履歴・プロフィール追加 | 様子見 | リテンション改善だが工数あり |
| G. Kill | 却下 | 検証前に早すぎる |

**結論** — **B + C + A** の軽微 pivot。ブランド「イケメンホストのソープコンサル」は維持。メンタルはサブ価値。

## 6. GTM 戦略

### チャネル

| 目的 | 施策 | KPI |
|---|---|---|
| 獲得 | note/X で「1 週間実験」記事（週 1）→ Hostorch 誘導 | 週 50 クリック |
| 獲得 | X スレッド + 固定ツイート | フォロワー 200、月間クリック 100 |
| 獲得 | 現役キャスト 5 名トライアル + Mom Test | 5 名完了 |
| 獲得 | SEO 記事 10 本（中長期） | 3 ヶ月後オーガニック 30/月 |
| 活性 | 例示質問を ICP 向け 8 件に絞る | 初回送信率 ≥ 40% |
| 活性 | 「1 週間後に教えて」を UI 表示 | 2 回目訪問率 ≥ 15% |
| 収益化 | Phase 1 はなし | — |

**やらない** — Product Hunt、プレス、一般向け広告、店舗営業

### プライシング

| Phase | モデル | 価格 |
|---|---|---|
| 1（〜3ヶ月） | 完全無料 | ¥0 |
| 2（PMF シグナル後） | フリーミアム | 無料 8 msg/日、有料 ¥980/月（無制限+履歴） |
| 3（スケール） | ティア拡張 | ¥1,980/月（高性能モデル+週次サマリー） |

### ローンチ

| フェーズ | 期間 | 内容 |
|---|---|---|
| Pre-launch | 1 週間 | Analytics、例示質問絞り込み、アンケート Form |
| Soft launch | 4 週間 | 5 名 + 匿名コミュニティ 1〜2、週次 FB |
| Public launch | 8 週間目〜 | note/X 公開、業界メディアへの自然露出 |
| Post-launch | 12 週間目〜 | Sean Ellis 判定、履歴 vs プロフィール、価格テスト |

## 9. リスク

| リスク | 影響 | 対策 |
|---|---|---|
| 無料 LLM 品質低下 | 高 | フォールバック維持、有料モデル検討 |
| プライバシー・バレ不安 | 高 | 履歴非保存を前面訴求、データ不使用明記 |
| OpenRouter 無料枠枯渇 | 中 | レートリミット強化、KV 日次上限 |
| 助言の法的・倫理リスク | 高 | 免責・拒否プロンプト維持、専門家エスカレーション |
| SNS 炎上・シャドウバン | 中 | 教育コンテンツ経由、露骨な広告回避 |
| 市場が小さすぎる | 中 | セグメント特化で深く取る |
